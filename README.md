## Getting Started

Install the module with: `npm install skeleton`

```js
var skeleton = require('skeleton');
skeleton.awesome(); // "awesome"
```

Install with cli command

```sh
$ npm install -g skeleton
$ skeleton --help
$ skeleton --version
```
